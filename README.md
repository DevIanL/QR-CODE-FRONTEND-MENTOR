# 📱 QR Code Component - Frontend Mentor

![Status](https://img.shields.io/badge/Status-Concluído-success)

## 💻 Sobre o Projeto
Solução para o desafio do componente de QR Code do [Frontend Mentor](https://www.frontendmentor.io/). O objetivo foi recriar o design proposto com a maior fidelidade possível, treinando habilidades de UI e responsividade.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3

## 🚀 Como Executar
1. Clone o repositório.
2. Abra no seu editor de código.
3. Execute através da extensão **Live Server**.
